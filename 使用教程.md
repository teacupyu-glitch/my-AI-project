## 一、准备 API 密钥（必须）

插件需要连接 DeepSeek 翻译服务，所以需要一个「钥匙」（API Key）。**这个钥匙是免费的，跟着下面做就行。**

### 第 1 步：打开 DeepSeek 官网

在浏览器打开：**https://platform.deepseek.com**

### 第 2 步：注册账号

点击右上角「注册」，用手机号或邮箱注册。已经有账号的直接登录。

### 第 3 步：获取 API 密钥

登录后：
1. 点击左侧菜单「API Keys」
2. 点击橙色按钮「创建 API Key」
3. 给它起个名字（随便写，比如"翻译插件"）
4. 点击「创建」
5. **立刻复制**出现的那串 `sk-xxxxxxxx` 开头的密钥

> ⚠️ 这串密钥**只会显示一次**，关闭后就看不到了。请马上复制并保存好！

### 第 4 步（重要）：充值 1 元

DeepSeek 虽然是付费 API，但翻译很便宜，**1 块钱够一个人用一年**。

1. 点击左侧「充值」
2. 充值金额选「1 元」
3. 用支付宝或微信付款

> 放心，花不完。翻译一整个网页大概几分钱。

## 二、电脑端使用

### Edge 浏览器安装

1. 打开 Edge 浏览器
2. 地址栏输入 `edge://extensions`，回车
3. 打开左下角的「**开发人员模式**」开关
4. 把 `ai-trans.zip` 文件**解压到一个文件夹**（右键 → 全部解压缩）
5. 回到 Edge，点击「**加载解压缩的扩展**」
6. 选择你刚解压出来的文件夹，点击「选择文件夹」
7. 完成！工具栏会出现一个翻译图标

### Chrome 浏览器安装

步骤基本一样：
1. 打开 Chrome，地址栏输入 `chrome://extensions`，回车
2. 打开右上角「**开发者模式**」
3. 点击「**加载已解压的扩展程序**」
4. 选择解压后的文件夹
5. 完成

### 配置插件

装好之后，需要把 API 密钥填进去：

1. 点击浏览器工具栏的**翻译插件图标**（如果没有，点拼图图标找一下，再点图钉固定）
2. 弹出窗口会提示「请先配置 API 密钥」
3. 点击「**设置**」，进入设置页面
4. 在「API 密钥」输入框里**粘贴你之前复制的 `sk-xxx` 密钥**
5. 点击「**保存 API 配置**」
6. 看到绿色提示「配置已保存」就 OK 了

其他设置可以不用动，默认已经是最优配置。如果想改翻译的目标语言（默认翻译成简体中文），在「翻译设置」里选一下保存即可。

### 翻译网页

配置好后，打开任何英文网页（或其他外语网页）：

1. 点击浏览器工具栏的翻译插件图标
2. 点击「**翻译当前页面**」
3. 等待几秒，译文会逐渐出现在原文下方
4. 页面上会出现一个**浮动控制栏**，可以：
   - 🔄 **显示原文/译文**：切换看原文还是译文
   - ↩ **撤销翻译**：恢复成翻译前的样子
   - ✏️ **编辑**：手动修改不满意的译文
   - ❌ **关闭**：隐藏控制栏

## 三、手机端使用（安卓）

苹果手机暂时不支持（iPhone 的浏览器不允许安装扩展插件）。

### 安装 Kiwi 浏览器

1. 打开手机上的**应用商店**（华为应用市场、小米应用商店等）
2. 搜索「**Kiwi Browser**」
3. 下载安装
4. 如果应用商店搜不到，去官网下载：在 Kiwi 里打开 `https://kiwibrowser.com`

### 导入插件

1. 把 `ai-trans-kiwi.zip` 文件传到手机上（微信发给自己、或者用数据线传）
2. 打开 Kiwi 浏览器，地址栏输入 `chrome://extensions`，回车
3. 打开「**开发者模式**」开关
4. 点击「**Load from ZIP**」或「从 ZIP 加载」
5. 在手机文件里找到 `ai-trans-kiwi.zip`，选中它
6. 完成

### 配置与使用

配置方法跟电脑端一样：
1. 在 Kiwi 里随便打开一个网页
2. 点击右上角**三个点** → 往下滑找到「AI 翻译助手」（或者在地址栏旁边找插件图标）
3. 点击进入设置，粘贴 API 密钥，保存
4. 之后打开英文网页，点插件图标 → 「翻译当前页面」即可

手机上翻译速度通常比电脑快（因为不走代理），实测一个英文网页 30 秒左右就能翻译完。

## 四、常见问题

### Q：翻译按钮点了没反应？
检查一下：
- API 密钥填了吗？（设置页确认一下）
- 密钥充钱了吗？（至少充 1 元）
- 当前网页是不是在排除列表里？（设置 → 排除网站）

### Q：翻译到一半报错了？
网络不稳定，**再点一次翻译按钮**就行。如果一直报错，检查 DeepSeek 账户余额。

### Q：想翻译的网站被跳过了？
设置 → 排除网站 → 把里面的域名删掉 → 保存。

### Q：翻译结果不对怎么办？
- 点控制栏的「编辑」按钮，手动改
- 设置 → 专有名词翻译 → 添加翻译对照表（比如 "Transformer" → "变换器"）

### Q：浏览器工具栏找不到插件图标？
- 点击工具栏的**拼图图标**（扩展程序）
- 找到「AI 翻译助手」
- 点旁边的**图钉**把它固定到工具栏

### Q：手机端翻译完页面卡住了？
翻译大型网页时手机内存可能不够，关掉网页重新打开，只翻译需要的部分。或者用电脑端翻译。

### Q：安全吗？我的 API 密钥会不会泄露？
- API 密钥存在你**自己的浏览器里**，不会上传到任何服务器
- 翻译请求直接从你的浏览器发给 DeepSeek，中间不经过第三方
- 放心使用
